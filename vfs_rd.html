Virtualization For Servers

    Red Hat 企業虛擬化建立在強大的核心虛擬機 (KVM) 管理程式和 oVirt 開放式虛擬管理平台上，專案由紅帽開啟並向社區發佈。Red Hat 企業虛擬化相較於專有虛擬化供應商而言，是虛擬 Linux 工作負載的首選平台，為尋求更佳總體擁有成本、更快投資回報率和避免供應商鎖定的企業組織提供了真正的策略性虛擬化替代。

    適用於大規模的虛擬化和內部雲部署

    Red Hat 企業虛擬化為企業提供了理想平台，此平台以大規模的虛擬化計畫和內部/私有雲部署為基礎。強大的核心虛擬機 (KVM) 系統管理程式，實現了創記錄的虛擬化基準結果以及無與倫比的整合率。功能齊全的企業管理系統使客戶能夠集中且有效地管理他們的整個虛擬環境（如虛擬資料中心、叢集、主機、客戶機虛擬伺服器、網路和儲存）。功能齊全、高可用性、即時遷移、儲存即時遷移、基於策略的負載平衡、影像管理、即時快照和精簡配置。

    Client作業系統支援Red Hat Enterprise Linux Client（32 和 64 位元）和 Microsoft Windows Client（32 和 64 位元），包括泛虛擬化網絡和block driver，Red Hat Enterprise Virtualization 已能支持整個IT架構。

    降低成本並大幅提昇利用率和靈活性

    將工作負載從物理伺服器轉換至虛擬機器，資料中心整合大幅提昇電腦資源的利用率和靈活性，且更能有效地利用能源和空間，同時降低操作成本。在伺服器硬體上節約的資金成本是您最大的收益，可將成本降低 40% 至 75%。另外，減少對空間和能源的需求也使營運更有利，且能改善伺服器持續管理。

    對於新伺服器硬體，由於多種工作負載（運行在虛擬機器上）能更有效地利用共用物理伺服器，工作事半功倍。另外，虛擬機器比物理伺服器能更快地進行配置。解決方案具備高可用性和快速恢復能力，比物理伺服器更能節約成本，有利於業務連續性。透過資源調度和策略驅動式負載均衡，優化了整個虛擬架構，可更有效地維護。

    Red Hat Enterprise Virtualization

    項目 功能
    Red Hat Enterprise Virtualization Hypervisor 基於Red Hat Enterprise Linux內核的裸機、高性能且安全的系統管理程序，採用了KVM (核心虛擬機) 技術
    影像化的輕量級系統管理程序具最小security footprint
    支援 VLAN、網卡綁定和廣泛的網路設備，包括10GB介面
    支援由Red Hat Enterprise Linux認證的所有64 位元 x86 伺服器平台。支援所有在Red Hat Enterprise Linux上已認證的儲存系統
    Red Hat Enterprise Virtualization Manager 集中管理系統具有搜索驅動圖形介面，支援數百台主機和數千台虛擬機器(以RHEL 6和JBoss EAP為基礎，實現高性能和可擴性)
    具有企業管理完整功能 即時遷移、儲存即時遷移、基於策略的負載均衡、高可用性、即時快照、節約能源、群集維護、影像管理、範本化、精簡配置、事件監控等
    業界領先的性能和可擴充性 主機支援最多160個內核和2 TB RAM
    客戶機支援最多160vCPU和2TB RAM；整合支援 200 台主機
    業界領先的SPECvirtSC2010結果
    支援 Windows 和 Linux 虛擬機器 伺服器：已認證的紅帽企業 Linux，RHEL 3、4、5、6；32 和 64 位元；Windows Server 2003、2003 R2、2008、2008 R2；32 和 64 位元SVVP 和 WHQL
    桌面：紅帽企業桌面 5 和 6，32 和 64 位元；Windows XP 32 位元：Windows 7 32 和 64 位元
    高階 SELinux kernel-based security 基於內核的最高等級內建安全性，能夠立即進行入侵偵測以及利用安全增強型Linux (SELinux) 和 sVirt 保護對虛擬機器和主機進行隔離
    自動化和客製化 RESTful API 能夠以程式設計方式自動運行、管理和配置紅帽企業虛擬化的各個方面
    基於 Python 的命令列介面允許腳本處理和自動化
    掛鉤機制使管理員能定義腳本以修改虛擬機器定義或運行系統命令
    自助服務 User Portal 使客戶能夠自己設置虛擬機器、定義範本和管理他們自己的環境
    管理員可為磁碟空間、CPU 使用和記憶體定義使用者配額
    報告和監控 基於JasperReports 的詳細記錄歷史能力與基礎產品相結合，旨在監控歷史使用情況、趨勢和服務品質。包括超過 25 個預置報告和儀錶板
    儲存 支援的儲存包括 iSCSI、Fibre Channel 、NFS、local storage、紅帽儲存和其他 POSIX 相容檔案系統
    支援的功能包括儲存即時遷移、即時快照、共用磁片、浮動磁片、VM 磁片熱插拔/拔出、LUN 直接連接至VM
    國際化 Administration 和 user portal 支援英語、中文、法語、德語、西班牙語、日語
